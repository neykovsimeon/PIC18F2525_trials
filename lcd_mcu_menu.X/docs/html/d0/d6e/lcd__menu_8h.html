Display type definitions. These are character LCD display type definitions.

LCD menu library support four types of character LCD display:

  • LCD 16 columns on 2 rows
  • LCD 16 columns on 4 rows
  • LCD 20 columns on 4 rows
  • LCD 20 columns on 2 rowsDefine display type 16x4.

◆ VISIBLE_MENU_HEADER

#define VISIBLE_MENU_HEADER   TRUE

Define whether menu header is always visible.

First row of the menu could be reserved for menu header.

Use flag TRUE to keep menu header always visible. If flag FALSE is used, then menu header will shift accordingly depending of the selected menu item.
